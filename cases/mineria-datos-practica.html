Bloque 2

Regresion lineal sobre datos sinteticos.

Con make_regression se construye un problema sencillo de una sola variable para entrenar una linea de regresion y evaluar el error cuadratico medio.

Bloque 3

Clasificacion con KNN.

El caso cambia de problema y pasa a un dataset 2D de clasificacion binaria. Antes de entrenar, normaliza con StandardScaler y luego evalua accuracy con k=5.

La parte fuerte aqui es que el mismo notebook incorpora preguntas sobre el efecto de cambiar K, lo que muestra comprension del tradeoff entre sensibilidad y suavizado.

Bloque 4

Aproximacion basica a ART2.

Sobre un subconjunto de 50 puntos se define una funcion propia para asignar clusters segun similitud por coseno y un umbral de vigilancia. Luego se compara el comportamiento con vigilancia 0.9, 0.7 y 0.4.

Idea principal

Si un punto no se parece lo suficiente al centro de un cluster, se crea uno nuevo. Con vigilancia alta aparecen mas grupos.

Valor conceptual

A diferencia de KMeans, aqui no se fija K desde el inicio. Eso conecta bien con la idea de clustering adaptativo.
