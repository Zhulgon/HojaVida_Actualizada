Visión por computador

MNIST y CIFAR-10 en un mismo flujo visual.

Este caso combina dos ejercicios sobre imagenes. El primero entrena una CNN para reconocer dígitos manuscritos en MNIST. El segundo extiende la práctica a CIFAR-10 filtrando tres clases visuales: airplane, automobile y truck, con una variante adicional apoyada en data augmentation.

2 ejercicios Dígitos manuscritos y clasificación visual con imágenes RGB.
TensorFlow Uso de Keras para definir, entrenar y evaluar redes convolucionales.
10 clases Salida softmax sobre MNIST para los números del 0 al 9.
3 clases CIFAR-10 filtrado en airplane, automobile y truck.

Resumen del caso

  • Pasa de una clasificación simple en grises a una variante multiclase con imágenes RGB.
  • Incluye preparación de datos, normalizacion, arquitectura, entrenamiento y lectura de predicciones.
  • Introduce una capa de augmentation para robustecer el segundo experimento.

Flujo

Cómo se organiza el desarrollo.

El archivo no se queda en una sola prueba. Avanza desde un problema clásico de visión por computador hasta una extensión más rica visualmente, lo que deja ver progresión y curiosidad técnica.

1

MNIST y salida multiclase

Se cargan los dígitos, se normalizan imágenes de 28 x 28 x 1 y se convierten etiquetas con to_categorical.

2

CNN base

Dos capas convolucionales con pooling, Flatten, capa densa y salida softmax para 10 clases.

3

Lectura visual de resultados

Se comparan imágenes del set de prueba con la etiqueta real y la predicción del modelo.

4

Extension a CIFAR-10

Se filtran tres clases, se remapean etiquetas y se repite el entrenamiento con y sin augmentation.

Experimentos

Qué demuestra cada bloque.

Aunque los dos modelos comparten una base CNN parecida, el caso deja ver que entiendes distintos tipos de entrada, salida y evaluación.

Bloque 1 - Dígitos manuscritos

Sirve para demostrar el manejo de una CNN clasica sobre imágenes pequeñas, una salida de diez clases y lectura de probabilidades con argmax.

model.fit(x_train, y_train, epochs=5, batch_size=64, validation_split=0.1)

Bloque 2 - Imágenes RGB filtradas

Amplia el caso hacia imágenes a color, uso de tres clases y una estrategia de augmentation con rotaciones, desplazamientos y volteo horizontal.

datagen = ImageDataGenerator(rotation_range=15, width_shift_range=0.1, height_shift_range=0.1, horizontal_flip=True)
