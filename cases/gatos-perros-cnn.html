Clasificacion binaria

Gatos vs perros con una CNN en grises.

Este ejercicio usa el dataset cats_vs_dogs desde TensorFlow Datasets, redimensiona imagenes a 100 x 100, las convierte a escala de grises y entrena una CNN binaria. Luego abre una segunda etapa de pruebas con imagenes cargadas manualmente para revisar predicciones reales fuera del dataset base.

Flujo

Preparacion de imagenes y entrenamiento.

El ejercicio recorre todo el proceso: inspeccion del dataset, transformacion, armado de arreglos, normalizacion y entrenamiento de la red.

1

Se cargan ejemplos desde cats_vs_dogs y se visualizan muestras.

2

Cada imagen se redimensiona, pasa a escala de grises y se guarda como tensor de 100 x 100 x 1.

3

Las imagenes se normalizan a valores entre 0 y 1 y las etiquetas se convierten a arreglo numerico.

4

Se entrena una CNN con tres bloques convolucionales y salida binaria.

Interpretacion

La clave: entender la sigmoid.

La salida del modelo representa probabilidad de pertenecer a una de las dos clases. Entender ese punto es importante para no leer mal una prediccion aparentemente baja o alta.

Lectura correcta

Si la salida esta cerca de 1, el modelo considera mas probable la clase perro. Si esta cerca de 0, la imagen se interpreta como gato con mayor confianza.

Ajuste en la confianza

El flujo corrige la presentacion de resultados usando 1 - prediccion para reportar confianza de gato cuando corresponde.

confianza = prediccion * 100 if etiqueta == "Perro" else (1 - prediccion) * 100

Pruebas reales

Predicciones sobre imagenes propias.

Este cierre saca al modelo del dataset base y lo pone a responder frente a imagenes cargadas manualmente, algo muy util para revisar generalizacion fuera del conjunto de entrenamiento.

Subida manual

Las imagenes se cargan con Colab, se transforman igual que el train y se pasan al modelo.

Prediccion individual

Se puede probar una imagen puntual como gato1.jpg o perro1.jpg para inspeccion directa.

Valor para portfolio

Muestra un paso extra de validacion pragmatica y capacidad de interpretar salidas de modelos binarios.
