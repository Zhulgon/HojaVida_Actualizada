Clasificacion binaria

Gatos vs perros con una CNN en grises.

Este notebook usa el dataset cats_vs_dogs desde TensorFlow Datasets, redimensiona imagenes a 100 x 100, las convierte a escala de grises y entrena una CNN binaria. Luego abre una segunda etapa de pruebas con imagenes cargadas manualmente para revisar predicciones reales fuera del dataset base.

100 x 100 Tamano unificado de entrada para las imagenes.
Escala de grises Reduccion a un solo canal para simplificar el modelo.
50 epocas Entrenamiento con validation_split=0.15.
Sigmoid Salida binaria para distinguir entre gato y perro.

Elementos clave del pipeline

  • Uso del dataset oficial de TensorFlow Datasets.
  • Conversor a escala de grises con OpenCV.
  • Pruebas con imagenes subidas manualmente al final del notebook.
TFDS OpenCV Binary classification

Flujo

Preparacion de imagenes y entrenamiento.

El notebook recorre todo el proceso: inspeccion del dataset, transformacion, armado de arreglos, normalizacion y entrenamiento de la red.

1

Se cargan ejemplos desde cats_vs_dogs y se visualizan muestras.

2

Cada imagen se redimensiona, pasa a escala de grises y se guarda como tensor de 100 x 100 x 1.

3

Las imagenes se normalizan a valores entre 0 y 1 y las etiquetas se convierten a arreglo numpico.

4

Se entrena una CNN con tres bloques convolucionales y salida binaria.

Interpretacion

La parte importante: entender la sigmoid.

El mismo notebook deja una aclaracion valiosa: al principio parecia raro ver "Gato con 0% de confianza", pero eso se corrige al entender que la salida representa probabilidad de ser perro, no una confianza simetrica entre dos etiquetas.

Lectura correcta

Si la salida esta cerca de 1, el modelo considera mas probable la clase perro. Si esta cerca de 0, la imagen se interpreta como gato con mayor confianza.

Ajuste en la confianza

El notebook termina corrigiendo la presentacion de resultados usando 1 - prediccion para reportar confianza de gato cuando corresponde.

confianza = prediccion * 100 if etiqueta == "Perro" else (1 - prediccion) * 100

Pruebas reales

Predicciones sobre imagenes propias.

Subida manual
