Análisis de audio

MFCC y espectro de frecuencias en Python.

Este ejercicio trabaja dos puntos: primero compara dos audios, guitarra y flauta, usando coeficientes MFCC y una distancia euclidiana entre sus promedios; luego analiza un audio adicional mediante FFT para leer la distribución de frecuencias y distinguir zonas graves y agudas.

2 instrumentos Comparación entre guitarra y flauta a partir de rasgos de timbre.
13 MFCC Extracción de coeficientes cepstrales para resumir características del audio.
FFT Transformada para observar contenido frecuencial de una señal.
0 - 5000 Hz Ventana de lectura para interpretar mejor el espectro resultante.

Que se trabaja aquí

librosa MFCC FFT
  • Comparación visual de coeficientes MFCC a lo largo del tiempo.
  • Resumen por medias para comparar el timbre promedio entre dos audios.
  • Uso de una ventana de Hann antes de calcular la FFT.

Primer bloque

Comparación de guitarra y flauta con MFCC.

El primer bloque usa librosa.feature.mfcc para obtener una representacion compacta del timbre de cada instrumento. Luego calcula un promedio por coeficiente para contrastar ambos audios de forma simple.

Flujo

1

Se cargan guitarra.wav y flauta.wav.

2

Se extraen 13 coeficientes MFCC para cada uno.

3

Se calcula la media por coeficiente y se grafican ambos perfiles.

4

Se usa distancia euclidiana para cuantificar la diferencia.

Lectura del resultado

Los MFCC no representan notas musicales de forma directa, sino características espectrales compactas. Por eso son útiles para distinguir timbres e identificar semejanzas o diferencias entre audios.

distancia = euclidean(mfcc_guitarra_mean, mfcc_flauta_mean)

Segundo bloque

Lectura del espectro de una señal con FFT.

El segundo bloque se enfoca en un audio llamado IA.wav. Primero aplica una ventana para evitar problemas en los bordes y luego usa la transformada de Fourier para observar la energía por frecuencia.

Ventana de Hann

Se multiplica la señal por una ventana suave para reducir artefactos al pasarla al dominio de frecuencia.

Frecuencias positivas

El gráfico conserva solo la mitad positiva del espectro, porque es la parte relevante para interpretar la señal real.

Guías visuales

Se agregan líneas en 300 Hz y 2000 Hz para ubicar zonas graves y agudas dentro del gráfico.
