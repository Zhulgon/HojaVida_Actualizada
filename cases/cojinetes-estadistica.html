Paso 2: Tamaño de la muestra

Calculo de n

Datos usados en la hoja

  • Nivel de confianza: 95%.
  • Factor probabilistico: Z = 1,96.
  • Margen de error: E = 0,1.
  • Desviacion estandar poblacional de un estudio anterior: θ = 0,4.
  • Tamaño de la poblacion: N = 420.

Formula desarrollada

n = (N · Z² · θ²) / ((N - 1) · E² + Z² · θ²)

A partir de esos valores se obtiene un resultado aproximado de 53,1302816, y en la hoja se deja finalmente n = 54 como tamaño de muestra.

Paso 3

t = (X̄ - μ) / (S / √n) t = (5.06433333 - 5.062) / (0.0124 / √54) t = 0.00233333 / 0.001686 t ≈ 4.351

Calcular el valor-P

ValorP = 2 (1 - A(Zcal)) A(Zcal) = 0.9999 ValorP = 1 - 0.9999 = 0.0001 Para nuestro caso: Valor-P = 0.9999

La hoja consigna que 0.9999 > 0.05, por lo tanto se acepta H0.

Regla de decision

Si Valor-P ≤ α, rechazamos H0. Si Valor-P > α, aceptamos H0.

Decision final

Como 0.9999 > 0.05, aceptamos H0
