# Caso de estadistica aplicada

Este caso replica en Python un ejercicio academico sobre diametros de cojinetes esfericos.

## Que incluye

- Tabla de frecuencias agrupadas
- Media, mediana y moda para datos agrupados
- Percentiles e intervalo intercuartilico
- Varianza, desviacion estandar y coeficiente de variacion
- Intervalos de confianza del 90% y 95% para la desviacion estandar poblacional usando chi-cuadrado

## Ejecutar

```bash
python cases/cojinetes_estadistica/analysis.py
```
