Parcial de inteligencia artificial

Dos experimentos en un mismo parcial.

Este archivo funciona como una actividad integradora. Primero entrena una CNN para reconocer digitos manuscritos en MNIST y luego extiende la practica a CIFAR-10 filtrando tres clases visuales para comparar entrenamiento normal contra un flujo con data augmentation.

3 clases CIFAR-10 filtrado en airplane, automobile y truck.

Lo mas valioso del parcial

  • Pasa de una clasificacion simple en grises a una variante multiclase con imagenes RGB.
  • Incluye preparacion de datos, normalizacion, arquitectura, entrenamiento y lectura de predicciones.
  • Introduce una capa de augmentation para robustecer el segundo experimento.

Flujo

Como esta armado el parcial.

El archivo no se queda en una sola prueba. Avanza desde un problema clasico de vision por computador hasta una extension mas rica visualmente, lo que deja ver progresion y curiosidad tecnica.

1

MNIST y salida multiclase

Se cargan los digitos, se normalizan imagenes de 28 x 28 x 1 y se convierten etiquetas con to_categorical.

2

CNN base

Dos capas convolucionales con pooling, Flatten, capa densa y salida softmax para 10 clases.

3

Evaluacion visual

Se comparan imagenes del set de prueba con la etiqueta real y la prediccion del modelo.

4

Extension a CIFAR-10

Se filtran tres clases, se remapean etiquetas y se repite el entrenamiento con y sin augmentation.

Experimentos

Que demuestra cada bloque.

Aunque los dos modelos comparten una base CNN parecida, el valor del archivo esta en mostrar que entiendes distintos tipos de entrada, salida y evaluacion.

Bloque 1 - Digitos manuscritos

Sirve para demostrar la comprension de una CNN clasica sobre imagenes pequenas, una salida de diez clases y lectura de probabilidades con argmax.

model.fit(x_train, y_train, epochs=5, batch_size=64, validation_split=0.1)

Bloque 2 - Imagenes RGB filtradas

Amplia el caso hacia imagenes a color, uso de tres clases y una estrategia de augmentation con rotaciones, desplazamientos y volteo horizontal.

datagen = ImageDataGenerator(rotation_range=15, width_shift_range=0.1, height_shift_range=0.1, horizontal_flip=True)
